Section 8.7: Polar Equations and Graphs

1. Polar Coordinate System

  • A point in the plane can be represented as (r, θ), where:
    • r: Distance from the origin (pole) to the point.
    • θ: Angle measured from the positive x-axis (polar axis).
  • Conversion between polar and rectangular coordinates:
    • x = r cos(θ)
    • y = r sin(θ)
    • r² = x² + y²
    • tan(θ) = y / x, where x ≠ 0

2. Plotting Points in Polar Coordinates

  • If r > 0, the point lies on the terminal side of θ.
  • If r < 0, the point is located in the opposite direction along the ray.

3. Alternative Forms for Polar Coordinates

  • The same point can have multiple polar representations:
    • (r, θ) = (-r, θ + π)
    • (r, θ) = (r, θ + 2πk), where k is any integer.

4. Conversion Between Polar and Rectangular Equations

  • To convert a rectangular equation x² + y² = a² into polar form:
    • Substitute x = r cos(θ) and y = r sin(θ)
    • r² = a² (which represents a circle)
  • To convert a polar equation into rectangular form, use:
    • r = f(θ) → x² + y² = f²(θ)

5. Graphs of Polar Equations

  • Circles: r = a (centered at the pole with radius a)
  • Lines: θ = constant (a line through the pole at angle θ)
  • Lemniscates: r² = a² cos(2θ) or r² = a² sin(2θ)
  • Cardioids: r = a(1 ± cos(θ)) or r = a(1 ± sin(θ))
  • Rose Curves: r = a cos(nθ) or r = a sin(nθ) (petal count depends on n)
    • If n is odd, number of petals = n
    • If n is even, number of petals = 2n
  • Limaçons: r = a ± b cos(θ) or r = a ± b sin(θ)
    • If |a/b| < 1, inner loop exists.
    • If |a/b| = 1, it forms a cardioid.
    • If |a/b| > 1, no inner loop.

6. Classification of Polar Equations

  • Symmetry tests:
    • Symmetric with respect to the polar axis (x-axis): Replace θ with -θ.
    • Symmetric with respect to the pole (origin): Replace r with -r.
    • Symmetric with respect to the vertical line θ = π/2: Replace θ with π - θ.

7. Important Tips for Graphing Polar Equations

  • Choose several values of θ, compute r, and plot the corresponding points.
  • Use symmetry to simplify graphing.
  • Identify the key features: loops, petals, and intercepts.

8. Examples

  1. Convert x² + y² = 9 to polar form:
    • r² = 9 → r = 3 (Circle of radius 3 centered at the pole).
  2. Convert r = 2 cos(θ) to rectangular form:
    • Multiply both sides by r: r² = 2r cos(θ)
    • Use r² = x² + y² and r cos(θ) = x
    • x² + y² = 2x → (x - 1)² + y² = 1 (A circle centered at (1,0) with radius 1).
